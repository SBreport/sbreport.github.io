# 키워드 조합기 웹앱

브라우저에서 바로 열어 쓰는 키워드 조합기입니다.

## 할 수 있는 일

- 지점, 시술, 키워드를 직접 입력할 수 있습니다.
- txt, md, csv, xlsx 파일에서 값을 불러올 수 있습니다.
- 조합 결과를 화면에서 바로 확인할 수 있습니다.
- 결과를 엑셀 또는 CSV 파일로 저장할 수 있습니다.

## 실행 방법

1. `keyword-combiner-web` 폴더를 엽니다.
2. [index.html](C:/Users/TUF%20A15%20FA507RM/OneDrive/%EB%B0%94%ED%83%95%20%ED%99%94%EB%A9%B4/codex/keyword-combiner-web/index.html) 파일을 브라우저에서 엽니다.
3. 또는 VS Code의 Live Server로 실행해도 됩니다.

## 입력 방식

- 직접 입력: 한 줄에 하나씩 입력
- 파일 불러오기: `txt`, `md`, `csv`, `xlsx`

## 파일 읽기 기준

- `txt`, `md`: 줄바꿈, 쉼표, 세미콜론 기준으로 읽습니다.
- `csv`: 첫 번째 열을 우선 읽고, 값이 없으면 표 전체에서 텍스트를 찾습니다.
- `xlsx`: 첫 번째 시트의 첫 번째 열을 우선 읽고, 값이 없으면 시트 전체에서 텍스트를 찾습니다.

예시:

```text
branch
강남
일산
다산
```

## 처음 사용할 때 추천 순서

1. `예시 채우기` 버튼 누르기
2. 조합 결과 미리보기 확인하기
3. 내 데이터로 바꾸기
4. 엑셀 또는 CSV로 저장하기

## 참고

- `xlsx` 불러오기와 엑셀 저장은 외부 SheetJS 스크립트를 사용합니다.
- 네트워크 상태에 따라 엑셀 기능이 일시적으로 불안정할 수 있습니다.
- 그런 경우에도 txt, md, csv 입력과 CSV 저장은 계속 사용할 수 있습니다.
