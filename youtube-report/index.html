API 키 발급 방법 (무료, 3분 소요)

구글 계정만 있으면 무료로 발급할 수 있습니다. 아래 링크를 순서대로 따라가세요.

1단계) Google Cloud Console 접속 → 상단 프로젝트 선택 → 새 프로젝트 → 이름 아무거나 입력 → 만들기

2단계) YouTube Data API v3 클릭 → 사용 버튼 클릭

3단계) 사용자 인증 정보 클릭 → + 사용자 인증 정보 만들기 → API 키 선택 → 생성된 키 복사

4단계) 복사한 키를 위 입력란에 붙여넣기 → 테스트 버튼으로 확인

💡 하루 10,000번 요청 가능하므로 보고서 용도로는 충분합니다.
⚠️ API 키는 비밀번호처럼 다른 사람에게 공유하지 마세요.

YouTube Analytics 연동 방법 (5~10분 소요)

위에서 만든 같은 Google Cloud 프로젝트에서 추가 설정하면 됩니다.

1단계) Analytics API 켜기
YouTube Analytics API 클릭 → 사용 버튼 클릭

2단계) 앱 정보 설정하기
Google Auth Platform 클릭 → 앱 이름, 이메일 입력
Audience → External 선택 → Testing 상태 유지
Test users에 분석할 유튜브 채널의 구글 계정 이메일 추가

3단계) OAuth Client ID 만들기
사용자 인증 정보 클릭 → + 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
애플리케이션 유형: 웹 애플리케이션 선택
승인된 JavaScript 원본에 현재 접속 주소 입력 (예: http://localhost)
생성된 클라이언트 ID를 복사 → 위 입력란에 붙여넣기

4단계) 연동하기
위의 🔗 Google 계정 연동 버튼 클릭 → 유튜브 채널 소유 계정으로 로그인

⚠️ origin_mismatch 오류가 나오면? → 3단계의 "승인된 JavaScript 원본"에 현재 브라우저 주소창의 주소가 정확히 들어가 있는지 확인하세요.
💡 Testing 상태에서는 7일마다 다시 연동해야 할 수 있습니다.

Anthropic API 키 발급 방법 (2분 소요)

1단계) Anthropic Console 가입 (구글 계정 가능)

2단계) API Keys 페이지 → Create Key 클릭 → 키 복사

3단계) 복사한 키를 위 입력란에 붙여넣기 → 테스트 버튼으로 확인

💡 신규 가입 시 $5 무료 크레딧이 제공됩니다. 보고서 1건당 약 $0.01 수준입니다.
⚠️ 무료 크레딧 소진 후에는 결제 수단 등록이 필요합니다.
