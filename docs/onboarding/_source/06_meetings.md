# 6. 회의 및 보고 가이드

## 내부 회의

- **정기 회의 없음** — 사안 발생 시 별도 미팅 진행
- 미팅 필요 시 사수 또는 이사에게 요청

## 클라이언트 보고

| 구분 | 주기 | 형식 |
|---|---|---|
| 주간 보고 | 주 1회 | 노션 온라인 서면 |
| 월간 보고 | 월 1회 | 노션 온라인 서면 |

> 보고 템플릿·양식은 내부 노션 링크 참조

## 보고 라인

```
부사수 → 사수 컨펌 → 진행
```

- 부사수는 작성 후 반드시 사수 컨펌을 거친 뒤 진행
- 사수는 컨펌 과정에서 피드백 및 수정 지시
- 사안에 따라 이사·대표까지 에스컬레이션
