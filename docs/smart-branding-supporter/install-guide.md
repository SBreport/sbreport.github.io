# 스마트 브랜딩 서포터 — 설치 및 사용 가이드

네이버 검색 결과를 분석해 **키워드 검색량**, **통합검색 카드 구성**, **광고 식별**, **블로그 강조**까지 한 패널에서 확인할 수 있는 Chrome 확장프로그램입니다. 마케팅 실무자가 검색 의도와 경쟁 환경을 빠르게 파악할 수 있도록 설계되었습니다.

**핵심 기능 5가지**

- 좌측 섹션 네비게이터 — 검색 결과의 영역·카드 구성을 한눈에
- 검색량 표시 — 현재 키워드의 PC/모바일 월간 검색량
- 블로그 결과 강조 — 블로그 카드를 색상으로 강조
- 글감 패널 숨기기 — 블로그 에디터의 우측 글감 패널 원클릭 숨김
- 블로그 글 전체 캡처 — 긴 블로그 글도 스크롤 영역 전체를 한 장 이미지로 저장 *(v0.5.0 신규)*

---

## 1단계: 파일 다운로드 및 압축 해제

1. 함께 전달된 **`smart-branding-supporter-v0.5.0.zip`** 파일을 컴퓨터에 저장합니다.
2. 저장된 ZIP 파일을 **마우스 오른쪽 클릭** → **압축 풀기**(또는 "여기에 풀기")를 선택합니다. (압축 푼 후 압축파일은 삭제!)
3. 압축을 풀면 **`smart-branding-supporter`** 폴더가 생깁니다. 이 폴더를 쉽게 찾을 수 있는 위치(예: 바탕화면)에 놓아두세요.

> **주의:** 이 폴더를 삭제하거나 이동하면 확장프로그램이 작동하지 않습니다. **한번 설치한 후에는 폴더 위치를 바꾸지 마세요.**

---

## 2단계: Chrome에 확장프로그램 설치

1. Chrome 브라우저의 주소창에 아래 주소를 입력하고 Enter를 누릅니다.

    ```
    chrome://extensions
    ```

2. 화면 **우측 상단**에 있는 **"개발자 모드"** 토글 스위치를 켭니다. (파란색으로 변하면 켜진 상태)

    [개발자 모드 토글 이미지 첨부]

3. 화면 **좌측 상단**에 나타나는 **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 클릭합니다.
4. 1단계에서 압축을 풀어둔 **`smart-branding-supporter` 폴더**를 선택하고 **"폴더 선택"** 을 클릭합니다.
5. 확장프로그램 목록에 **"스마트 브랜딩 서포터 v0.5.0"** 이 나타나면 설치 완료입니다.

---

## 3단계: 확장프로그램 아이콘 고정

설치 직후에는 아이콘이 숨겨져 있을 수 있습니다. 쉽게 접근하려면 아래처럼 고정하세요.

1. Chrome 우측 상단의 **퍼즐 모양 아이콘** (확장프로그램 메뉴)을 클릭합니다.
2. **"스마트 브랜딩 서포터"** 옆의 **핀(📌) 아이콘**을 클릭합니다.
3. 이제 주소창 옆에 **파란색 전구 아이콘** 💡 이 항상 표시됩니다.

    [전구 아이콘 고정 이미지 첨부]

## 사용 방법

### 검색 결과 분석 (메인 기능)

1. 네이버에서 원하는 키워드를 검색합니다. (예: "청주 피부과")
2. 좌측에 **"스마트 브랜딩 서포터"** 박스가 자동으로 표시됩니다.

    [네이버 검색 + SBS 박스 표시 이미지 첨부]

3. 박스에서 다음 정보를 한눈에 확인할 수 있습니다.

| 영역 | 설명 |
| --- | --- |
| 검색량 (월간) | 현재 키워드의 PC / 모바일 / 합계 검색량 + 경쟁도 |
| 섹션 네비게이터 | 통합검색·플레이스·블로그·인플루언서 등 영역 목록 |
| 카드 배지 | `[블]` 블로그 · `[카]` 카페 · `[지]` 지식인 · `[광]` 광고 · `[웹]` 웹사이트 |
| 블로그 카운트 | "(5건, 블 2)" 형식으로 영역 내 블로그 개수 즉시 인지 |
| 푸터 CTA | "키워드 전문 분석하기 →" 클릭 시 정식 분석 사이트로 이동 |

### 블로그 결과 강조

검색 결과 중 **블로그 카드만** 파란색(또는 선택 색상)으로 배경 강조되어 시각적으로 즉시 식별됩니다. 네이버 인플루언서 콘텐츠도 함께 강조됩니다.

### 글감 패널 숨기기

네이버 블로그 **에디터(글쓰기 화면)** 의 우측 글감 패널(책·음악·영화·쇼핑 등)이 자동으로 숨겨집니다. 별도 클릭 불필요.

### 옵션 설정 (전구 아이콘 클릭)

Chrome 우측 상단의 **파란색 전구 아이콘** 💡 을 클릭하면 popup이 열리고 기능별 토글을 ON/OFF 할 수 있습니다.

| 기능 | 기본값 |
| --- | --- |
| 글감 패널 숨기기 | ON |
| 좌측 섹션 네비게이터 | ON |
| 블로그 결과 강조 | ON (색상: 녹 / 노 / 하늘 선택 가능) |
| 검색량 표시 | ON |
| 연관 검색어 표시 | OFF (필요 시 켜기) |

## 자주 묻는 질문

**Q. 박스가 안 보여요.**
→ 네이버 검색 결과 페이지(`search.naver.com`)에서만 작동합니다. 검색을 한 번 한 뒤 페이지를 새로고침(F5)해 보세요. popup에서 "좌측 섹션 네비게이터" 토글이 ON인지도 확인해 주세요.

**Q. 검색량이 표시되지 않아요.**
→ popup에서 "검색량 표시" 토글이 ON인지 확인해 주세요. 일시적 네트워크 오류라면 페이지 새로고침으로 해결되는 경우가 많습니다.

**Q. "베타 버전이 종료되었습니다" 가 떠요.**
→ 베타 테스트 기간이 종료되었습니다. "공지 확인" 버튼을 누르면 정식 버전 안내 페이지로 이동합니다.

**Q. 확장프로그램을 제거하고 싶어요.**
→ `chrome://extensions` 에서 "스마트 브랜딩 서포터"의 **"삭제"** 버튼을 클릭하면 됩니다.

**Q. Chrome 업데이트 후 작동하지 않아요.**
→ `chrome://extensions` 에서 확장프로그램의 **새로고침 아이콘(🔄)** 을 클릭해 보세요.

**Q. 어떤 검색어에서 특정 영역이 안 잡혀요.**
→ 피드백 채널로 검색어와 함께 알려주시면 다음 버전에서 보강하겠습니다.
