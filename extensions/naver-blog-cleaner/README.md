# 네이버 블로그 글감 숨기기 확장프로그램

네이버 블로그 에디터 하단의 **글감 패널**을 원클릭으로 숨기거나 다시 보이게 하는 Chrome 확장프로그램입니다.

## 설치 방법

1. 이 폴더를 컴퓨터에 다운로드/압축 해제합니다.
2. Chrome 브라우저에서 `chrome://extensions` 페이지로 이동합니다.
3. 우측 상단의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다.
5. 다운로드한 폴더(`naver-blog-cleaner`)를 선택합니다.

## 사용 방법

- 네이버 블로그 에디터 페이지에서 **확장프로그램 아이콘(녹색 X)**을 클릭하면 글감 패널이 숨겨집니다.
- 다시 클릭하면 글감 패널이 다시 나타납니다.

## 자동 숨기기 (선택사항)

페이지 로드 시 글감을 자동으로 숨기고 싶다면 `content.js` 파일 하단의 주석 처리된 자동 숨기기 코드의 주석을 해제하세요.

## 참고사항

- 네이버 블로그 에디터의 DOM 구조가 변경되면 동작하지 않을 수 있습니다.
- 문제가 발생할 경우 확장프로그램을 비활성화하거나 제거하시면 됩니다.
