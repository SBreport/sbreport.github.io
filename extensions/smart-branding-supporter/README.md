# 스마트 브랜딩 서포터 (SBS)

네이버 블로그·검색 분석용 통합 Chrome 확장.

## 기획서
`기획/기획서.md` 참조.

> 참고: Chrome 확장은 `_`로 시작하는 폴더를 예약어로 취급하므로 폴더명은 `기획`(언더스코어 없음)으로 유지.

## 현재 상태
v0.1.0 — Phase 0 (골격 단계). 실제 기능 로직은 비어있음.

## 로컬 설치 (개발 중)
1. Chrome → `chrome://extensions/`
2. 우상단 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램 로드" → 이 폴더(`smart-branding-supporter`) 선택
