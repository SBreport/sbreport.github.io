# YouTube Script Copy 확장프로그램

유튜브 영상의 **메타데이터 + 자막(스크립트)** 을 클릭 한 번으로 클립보드에 복사하는 Chrome 확장프로그램입니다.

## 주요 기능

- 영상 제목·채널·길이·업로드일 등 메타데이터 자동 추출
- **InnerTube API 우선 사용** (Glasp 방식). 실패 시 자막 패널 DOM 스크래핑으로 폴백
- 자막을 **문단 단위로 자동 분리**:
  - 300자 이상 + 문장 종결 → 문단 분리
  - 800자 초과 시 강제 분리
  - 10초 이상 침묵 구간 → 주제 전환으로 간주, 강제 분리
- 타임스탬프 자동 포맷팅

## 설치 방법

1. 이 폴더를 다운로드/압축 해제합니다.
2. Chrome에서 `chrome://extensions` 페이지로 이동합니다.
3. 우측 상단 **개발자 모드** 활성화.
4. **압축해제된 확장 프로그램을 로드합니다** 클릭.
5. `youtube-script-copier` 폴더 선택.

## 사용 방법

1. 유튜브 영상 페이지로 이동.
2. 페이지에 추가된 **YT Script Copy 버튼** 클릭.
3. 메타데이터와 스크립트가 자동으로 클립보드에 복사됩니다.

## 기술 사양

- Manifest V3
- 권한: 없음 (clipboard write via content script)
- 적용 도메인: `youtube.com`
- 의존성: 없음 (vanilla JavaScript)

## 참고사항

- 유튜브 DOM 구조 변경 시 동작이 불안정해질 수 있습니다.
- 자막이 없는 영상에서는 메타데이터만 복사됩니다.
- 자동 생성된 자막과 직접 제작한 자막 모두 지원합니다.
