📖 설치 방법

  1. 다운로드한 ZIP을 원하는 폴더에 압축 해제 ⚠️ 압축 해제한 폴더는 사용 중 삭제 X — 그대로 둬야 작동
  2. Chrome 주소창에 입력: chrome://extensions/
  3. 우상단 개발자 모드 토글을 ON
  4. 좌상단 "압축해제된 확장 프로그램을 로드합니다" 클릭 → 1번 폴더 선택
  5. 확장 목록에 "스마트 브랜딩 서포터"가 나타나면 완료 toolbar 우상단에 파랑 전구 아이콘 표시됨
